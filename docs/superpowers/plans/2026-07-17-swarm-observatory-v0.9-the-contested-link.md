## Sequencing rationale

W1 first because all three benches flagged the same live wound and the comms lens
inherits the register (a lens built first would mint states with no voice to
wear); W2 early because every later wave adds chrome; W3 before everything
downstream because the lenses and the table consume the decoders; W4 before W5/W6
because the reveal clock is their shared spine; W5 before W6 because the hero
anchors the cycle; W7 after the lenses so the table lists comms/track kinds on
day one; W8 last because it is presentation-pure and the owner sees it against
finished stages. The launch beat rides the endgame rituals.
